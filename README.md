BeaconDetection
====

beacon技術と触れ合うためのアプリを目指しています。

### Beacon検知

* 起動するとビーコンを探します
* 取得した情報を画面に表示します

## Future

* 位置情報をトリガーにデータ取得
* Beacon化機能
* Beacon複数取得表示
* レイアウトをいい感じに
* Const.swiftに設定している値を変更可能に

## Requirement

* Xcode 9.0
* Swift 4
* Carthage 0.25.0

## Install

```
$ cd BeaconDetection/
$ carthage update --platform iOS
```
