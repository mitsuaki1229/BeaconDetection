BeaconDetection
====

beacon技術と触れ合うためのアプリを目指しています。

### Beacon検知

* 起動するとビーコンを探します
* 取得した情報を画面に表示します

### 検知データ送信

* 指定したサーバに検知データ送信
+ Const.swiftに送信先や端末識別番号を記載して下さい。

## Future

* 位置情報をトリガーにデータ取得
* Beacon化機能
* Beacon複数取得表示
* レイアウトをいい感じに
* Const.swiftに設定している値を変更可能に

## Requirement

* Xcode 8.3.3
* Swift 3.1
* Carthage 0.24.0

## Install

```
$ cd BeaconDetection/
$ carthage update --platform iOS
```
