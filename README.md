BeaconDetection
====

Beacon experience.

## Requirement

* Xcode 9.0
* Swift 4
* Carthage 0.25.0

## Install

```
$ cd BeaconDetection/
$ carthage update --platform iOS
```

## Usage

* DetectionView
 - Search and display Beacons info.
* Simulator
 - Instration App is become Beacon.
